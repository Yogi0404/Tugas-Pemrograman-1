#  Project Dasar Pemrograman Java

## Deskripsi

Project ini berisi kumpulan latihan dan tugas dalam mempelajari dasar-dasar pemrograman menggunakan bahasa **Java**. Materi yang dipelajari masih berada pada tahap fundamental, seperti penggunaan variabel, input/output, serta struktur kontrol.

Project ini disusun berdasarkan pertemuan perkuliahan, sehingga setiap folder merepresentasikan materi yang dipelajari pada tiap pertemuan.

## Tentang Java

Java adalah bahasa pemrograman yang banyak digunakan untuk mengembangkan berbagai jenis aplikasi, mulai dari desktop, web, hingga mobile. Java dikenal dengan konsep *platform-independent*, yang memungkinkan program dijalankan di berbagai sistem operasi menggunakan Java Virtual Machine (JVM).

## Materi yang Dipelajari

Beberapa konsep dasar Java yang terdapat dalam project ini antara lain:

* Variabel dan tipe data
* Operator (aritmatika, logika, perbandingan)
* Input dan Output (menggunakan Scanner)
* Struktur kontrol (if-else)

## Struktur Project

Struktur folder dalam project ini disusun sebagai berikut:

* **Pertemuan X/**

  * **libraries/** → Berisi library yang digunakan (jika ada)
  * **source package/**

    * **tugas/** → Berisi file tugas yang diberikan
    * **latihan/** → Berisi latihan selama pembelajaran

Struktur ini dibuat untuk memisahkan antara latihan dan tugas, serta mempermudah dalam mengikuti alur pembelajaran tiap pertemuan.

## Cara Menjalankan Program

1. Pastikan Java sudah terinstall di komputer
2. Buka folder project
3. Compile file Java:

   ```bash
   javac NamaFile.java
   ```
4. Jalankan program:

   ```bash
   java NamaFile
   ```

## Tujuan Project

Tujuan dari pembuatan project ini adalah:

* Memahami dasar-dasar pemrograman Java
* Melatih logika pemrograman
* Membiasakan penulisan kode yang terstruktur
